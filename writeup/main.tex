\subsection{Probability Theory and Stochastic Processes}

The definitions and notation in this section follow \textcite{shreve2004}, Chapters 1--3.

First, the probability measure.

\begin{definition}
Let \(\Omega\) be a nonempty set, \(\mathcal{F}\) be a \(\sigma\)-algebra of subsets of \(\Omega\), and \(\mathbb{P}\) be a probability measure over \(\mathcal{F}\). The triple \((\Omega, \mathcal{F},\mathbb{P})\) is called a \textup{probability space}.
\end{definition}

\begin{definition}
Let \((\Omega, \mathcal{F},\mathbb{P})\) be a probability space. If a set \(A \in \mathcal{F}\) satisfies \(\mathbb{P}(A) = 1\), we say that the event A occurs \textup{almost surely}.
\end{definition}

We next formalise the concept of a stochastic process, the mathematical object used to model the random evolution of quantities such as asset prices.

\begin{definition}
Let \(\Omega\) be a nonempty set. Let T be a fixed positive number, and assume that for each \(t \in [0, T]\), there is a \(\sigma\)-algebra \(\mathcal{F}(t)\). Assume further that if \(s \leq t\), then every set in \(\mathcal{F}(s)\) is also in \(\mathcal{F}(t)\). Then we call the collection of \(\sigma\)-algebras \(\mathcal{F}(t)\), \(0 \leq t \leq T\), a \textup{filtration}.
\end{definition}

\begin{definition}
Let \(\Omega\) be a nonempty sample space equipped with a filtration \(\mathcal{F}(t)\), \(0 \leq t \leq T\). Let \(X(t)\) be a collection of random variables indexed by \(t \in [0, T]\). We say this collection of random variables is an \textup{adapted stochastic process} if, for each t, the random variable \(X(t)\) is \(\mathcal{F}(t)\)-measurable.
\end{definition}

In the context of continuous-time mathematical models, the flow of public information is interpreted as a filtration. The models adapt asset prices, portfolio processes, and wealth processes to this filtration.

\begin{definition}
Let \((\Omega, \mathcal{F},\mathbb{P})\) be a probability space, let T be a fixed positive number, and let \(\mathcal{F}(t)\), \(0 \leq t \leq T\), be a filtration of sub-\(\sigma\)-algebras of \(\mathcal{F}\). Consider an adapted stochastic process \(M(t)\), \(0 \leq t \leq T\). If
    
    \[
    \mathbb{E}[M(t) \mid \mathcal{F}(s)] = M(s) \text{ for all } 0 \leq s \leq t \leq T,
    \]

we say this process is a \textup{martingale}. It has no tendency to rise or fall.

\end{definition}

\subsection{Brownian Motion}

\begin{definition}
Let \((\Omega, \mathcal{F},\mathbb{P})\) be a probability space. For each \(w \in \Omega\), suppose that there is a continuous function \(W(t)\) of \(t \geq 0\) that satisfies \(W(0) = 0\) and that depends on \(w\). Then, \(W(t)\), \(t \geq 0\), is a \textup{Brownian motion} if for all \(0 = t_0 < t_1 < \cdots < t_m\) the increments
    \begin{equation}
    W(t_1) - W(t_0),\; W(t_2) - W(t_1),\; \ldots,\; W(t_m) - W(t_{m-1})
    \end{equation}

    are independent and each of these increments is normally distributed with

    \begin{equation}
    \mathbb{E}[W(t_{i+1}) - W(t_{i})] = 0
    \end{equation}
    \begin{equation}
    \mathrm{Var}[W(t_{i+1}) - W(t_{i})] = t_{i+1} - t_i
    \end{equation}

\end{definition} 

We will now highlight two key properties of Brownian motion.

\begin{theorem}
Brownian motion is a martingale.
\end{theorem}

\begin{proof}
Let \(0 \leq s \leq t\) be given. Then
    \begin{align*}
    \mathbb{E}[W(t) \mid \mathcal{F}(s)] &= \mathbb{E}[W(t) - W(s) + W(s) \mid \mathcal{F}(s)] \\
                                         &= \mathbb{E}[W(t) - W(s) \mid \mathcal{F}(s)] + \mathbb{E}[W(s) \mid \mathcal{F}(s)] \\
                                         &= \mathbb{E}[W(t) - W(s)] + W(s) \\
                                         &= W(s).
    \end{align*}
The second equality is linearity of conditional expectation. For the third: \(W(t) - W(s)\) is independent of \(\mathcal{F}(s)\), so conditioning drops; \(W(s)\) is \(\mathcal{F}(s)\)-measurable, so \(\mathbb{E}[W(s)\mid\mathcal{F}(s)] = W(s)\). The last equality uses \(\mathbb{E}[W(t)-W(s)] = 0\). See \cite[p.~98]{shreve2004}.
\end{proof}


\begin{theorem}
Let W be a Brownian motion, and let \([W, W](T)\) denote the quadratic variation of W up to time T. Then \([W, W](T) = T\) for all \(T \geq 0\) almost surely.
\end{theorem}

\begin{proof}
See \cite[p.~102]{shreve2004}.
\end{proof}

Brownian motion is the natural model for the random component of asset prices: its continuous paths, stationary independent increments, and Gaussian distribution are consistent with observed log-returns over short time intervals. The quadratic variation result \([W,W](T) = T\) is particularly important, as it encodes the heuristic \((\mathrm{d}W)^2 = \mathrm{d}t\) that drives Itô's formula in the next section.
